About Me

Hello! My name is Victor. I am a multi-disciplinar engineer whose interest in web development started a few years ago when I was involved in the back-end development of a monitoring app at my old job.

After that experience, I decided I wanted to learn more about web development so I started an online course on full stack development in Ruby on Rails. Ever since then, I have been constantly learning and developing my personal projects. Some technologies I've been working with are:

  • HTML, CSS
  • Javascript (ES6+)
  • React, Gatsby
  • Node.js, Express
  • Ruby on Rails
  • Python
  • SQL, MongoDB

A web app made in React for creating a "watch later" playlist of Twitch videos. The app lets the user paste the URL of the video and, once it is parsed and verified, the video title can be customized and finally added to the playlist. Videos are embedded as interactive frames when played.
